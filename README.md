# Eternal UI LIB

inimal black and white Roblox UI library.

## Features

• Modern design
• Minimal layout
• Rounded corners
• Black and white theme
• Tabs
• Buttons
• Toggles

## Installation

```lua
local UI =
loadstring(game:HttpGet("https://raw.githubusercontent.com/norixu/eternal-ui/main/source.lua"))()
```

## Example

```lua
local Window =
UI:CreateWindow({
Title = "Eternal UI"
})
```
